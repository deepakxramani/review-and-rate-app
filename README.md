# Review & Rate App

A full-stack MERN application that enables users to discover companies, submit reviews, rate businesses, and explore company profiles through a clean and responsive interface.

## ✨ Features

* Add, update, and delete companies
* Upload company logos
* Browse company listings with search, filtering, sorting, and pagination
* View detailed company profiles
* Submit reviews with ratings
* Automatic average rating calculation
* Like reviews
* Responsive user interface
* RESTful API with versioning (`/api/v1`)

## 🛠️ Tech Stack

### Frontend

* Next.js 15
* TypeScript
* Redux Toolkit
* SCSS Modules
* Axios
* React Hook Form

### Backend

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose
* Multer
* Express Validator

## 📂 Project Structure

```text
review-and-rate-app/
├── frontend/
└── backend/
```

## 🚀 Getting Started

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Create the required `.env` files for both frontend and backend before running the application.
